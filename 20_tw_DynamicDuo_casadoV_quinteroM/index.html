Why Tailwind?

Tailwind provides an alternative for CSS inline or in a CSS file. It allows for more cohesive and readable code as you can see the styling of each tag easier. Tailwind's strongest feature is also its customizability as it allows users to create their own tags for ease of use.




Putting Tailwind into your project

Paste 
<script src="https://cdn.tailwindcss.com"></script>
into your HTML head tag




Using Tailwind

To implement Tailwind into an HTML tag, place class="" inside the beginning tag. Inside the quotes, write one of the built-in methods from tailwind's documentation




Custom Configs

Tailwind also allows you to implement your own definitions for CSS. For example, inside of the HTML for this web page, we defined the color "clifford" with the hex code '#da373d'.
