We got this using the GET request

The GET request method appends information in the URL of the website. This allows users to see inputted info in browsing history, and to bookmark specific inputs.
It also has a maximum input size, and can possibly cause issues with special characters in the url.

The POST request method appends the information to the HTTP request, which is what the browser sends when it attempts to load a website.
This prevents data from being saved when the page is reloaded. This also prevents users from seeing the data in the URL.
